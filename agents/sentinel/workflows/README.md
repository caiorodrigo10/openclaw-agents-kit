|---|---|---|
| Resposta a incidente | `incidente.md` | Servico caiu ou esta com problema |
| Deploy novo servico | `deploy-novo-servico.md` | Adicionar novo servico ao servidor |
| Relatorio de status | `relatorio-diario.md` | Gerar resumo do estado da infra |
