# Scripts - Indice

Scripts uteis para automacao. Requerem acesso SSH ao servidor.

| Script | Arquivo | Funcao |
|---|---|---|
| Healthcheck | `healthcheck.sh` | Verificacao rapida de todos os servicos |
| Coleta de metricas | `coleta-metricas.sh` | Coleta CPU/memoria/disco para relatorio |

Nota: esses scripts devem ser executados no host (via SSH), nao dentro do container OpenClaw.
